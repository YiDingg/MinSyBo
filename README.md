# MinSyBo (Minimum System Board of STM32F103C8T6)
### 1.项目说明
这是一个STM32F103C8T6最小系统板，原理图和PCB使用立创EDA专业版进行设计，相关文件全部开源。

设计思路与具体细节见文章：[📚 STM32F103C8T6最小系统板设计](https://www.writebug.com/article/a7f918f4-1140-11ef-bb3d-0242ac1b0018)
